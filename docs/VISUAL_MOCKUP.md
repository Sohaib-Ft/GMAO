## Interactive States

### Hover State
When hovering over a work order card:
- Background changes from gray-50 to gray-100
- Arrow icon (â†’) appears on the right side
- Cursor changes to pointer
- Subtle transition animation

### Click Action
Clicking on a work order:
- **Employees**: Navigate to `/workorders` (index page)
- **Technicians**: Navigate to `/technicien/workorders/{id}` (detail page)
- **Admins**: Navigate to `/admin/workorders/{id}` (detail page)

## Responsive Behavior

### Desktop (> 1024px)
- Profile info on left (1/3 width)
- Forms on right (2/3 width)
- Work orders card shows 5 items

### Tablet (768px - 1024px)
- Profile info on left (1/3 width)
- Forms on right (2/3 width)
- Work orders card shows 3-5 items

### Mobile (< 768px)
- Stacked layout (full width)
- Profile info on top
- Work orders card below
- Forms at bottom
- Shows 3 items
